# Функция *parseSnippets*
## Назначение функции
Функция для загрузки шаблонов кода (сниппетов) в формате 1С:Предприятия

## Параметры функции
* **stData** - *string*, текстовое содержимое файла шаблона .st
	
## Пример вызова
```javascript
parseSnippets('{0,{"Пока",0,0,"Пока","Пока <?""Условие""> Цикл......');
```
